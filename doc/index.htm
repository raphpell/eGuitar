Objets Javascript de l'application eGuitar

Tous les objets vues sont dépendant de l'objet Config. Soit :

  • Objet vue principal prototype Manche
  • prototype IntervalBox : Affichage d'intervalles.
  • prototype HarmonieForm : Affichage de choix d'intervalles.
  • prototype HarmonieTable : Affichage d'accords présent dans des intervalles.
