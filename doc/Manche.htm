Class Manche

Affichage par défaut

Toutes les configuration sont globales. Elles sont stockées et réutilisées lors du chargement de page.

Un changement entraine une répercution sur toutes les instances courantes et futures.

Configuration partagées:

  • la3 : Nombre - fréquence du La. Valeur par défaut 440hz.
  • lefthanded : Booléen - Joueur gauché.
  • mask : Chaîne - masque des notes affiché.
  • mirror : Booléen - reflet du manche dans un mirroir.
  • notation : Tableau - Choix de la notation : La ou A, avec ♭ ou ♯.
  • notes : Booléen - Affichage du nom des notes du manche.
  • numbers : Booléen - Affichage des numéros des cases.
  • octaves : Booléen - Affichage simplifié des octaves.
  • scale : ...
  • sound : Booléen - sons associés aux notes. Experimental.
  • tonic : Note - tonique de la 'gamme' ( Points rouge non désélectionnable )
  • tuning : Nombre - 'Accordage du manche'.

Affichage : Utilisation recommandée

On partage les préférences d'affichage d'un utilisateur entre toutes les instances.
Le reste doit appartenir à une et une seule instance.
